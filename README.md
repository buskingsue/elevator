# Elevator Project

<img width = "40%" src="https://github.com/buskingsue/elevator/blob/master/elevator.png">

## 1. 기능

- 1층 2층 이동
- LCD 디스플레이에 상태 정보 표시, 현재 층 표시
- 특정 층에 위치한 포토 인터럽트에 도착 시 정지


## 2. 사용한 부품

- Nucleo F411RE
- 택트 스위치 2개
- 포토 인터럽터 2개
- SG90 서보모터
- LCD 디스플레이
